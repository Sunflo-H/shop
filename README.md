# Project : 쇼핑몰 프로젝트
# Stack
- HTML
- CSS
- React

# API
- Firebase
- cloudinary

# Library
- React Query
- React Router 
- SweetAlert

# 목표
상품을 보여주고, 장바구니에 담는 등의 기능에 OAuth 로그인, 그리고 핫한 AI를 이용한 기능을 만들고자 합니다.

# 기능
- 모든 등급의 회원들이 사용 가능한 기능
  - 상품리스트
  - 장바구니
  - 찜하기
  - AI 추천
- 어드민 등급만 가능한 기능
  - 상품등록
